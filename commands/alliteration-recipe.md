# Alliteration Recipe Command

Goal: create a recipe where the ingredient names mostly start with the same letter.

The first command argument is the target letter.

When this command is used:

- treat the first argument as the target letter
- if no letter is provided, choose one sensible letter and state it
- create a recipe with a dish name and ingredient list where ingredient names mostly start with that letter
- prefer a recipe that sounds coherent and edible over perfect constraint satisfaction
- if a perfect alliterative version is impossible, keep exceptions minimal and call them out briefly
- keep the recipe concise and fun
